 House Prices Preprocessing Project

هذا المشروع هو التكليف الأول لمادة تعلم الآلة، ويتضمن معالجة البيانات واستكشافها.
This project is the first assignment for the Machine Learning course, focusing on data preprocessing and EDA.

 Steps (الخطوات):
- Handling missing values (معالجة القيم المفقودة).
- Log transformation for SalePrice (التحويل اللوغاريتمي للسعر).
- Feature Engineering: TotalSF (هندسة الميزات).
